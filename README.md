# rust-ddp
A Rust Library to connect to ddp servers and talk to Meteor servers.

# New in v0.0.2

Addition of basic support for MongoCollections!

 - Insert
 - Remove
 - Upsert
 - Update
 - *Inmove*
 - *Resert*

Much less warnings!

Better method APIs!
